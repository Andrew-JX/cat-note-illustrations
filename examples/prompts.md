# Prompt Examples

## 单个观点生成一张图

```text
Use $cat-note-illustrations 为“AI 不是替你思考，而是帮你把思考摊开检查”生成一张正文配图。
```

## Context Engineering

```text
Use $cat-note-illustrations 为“context engineering：不是把所有东西塞进 prompt，而是选择正确的目标、约束和资料”生成一张正文配图。
```

## Harness Engineering

```text
Use $cat-note-illustrations 为“harness engineering：用输入、检查和输出反复测试 AI 行为”生成一张正文配图。
```

## Loop Engineering

```text
Use $cat-note-illustrations 为“loop engineering：通过运行、观察、调整、再跑来改进 AI 工作流”生成一张正文配图。
```

## 飞书笔记配图规划

```text
Use $cat-note-illustrations 先不要生成图片。
请为下面这篇飞书笔记设计 4 张正文配图方案。
每张图写清楚：放在哪段后、主题、核心意思、结构类型、猫猫动作、建议中文标注词。

<粘贴正文>
```

## 按方案生成

```text
Use $cat-note-illustrations 按上面的方案生成图片。
```

## 生成更少文字的版本

```text
Use $cat-note-illustrations 重新生成上一张图。
要求：保留原意思，减少中文标注，最多 2 个短词。
```

## 修正猫猫角色

```text
Use $cat-note-illustrations 修改这张图。
要求：保持构图和含义，把猫猫改回默认 Specimen 0 主角。
```
