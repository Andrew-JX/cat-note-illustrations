# Contributing

谢谢你愿意改进 Cat Note Illustrations。

这个项目的目标不是收集很多猫猫风格，而是让一个固定的猫猫 IP 稳定服务中文文章、飞书笔记和知识型内容配图。提交改动时，请优先保护这种一致性。

## 可以贡献什么

- 更稳定的中文 prompt 示例
- 更清晰的 shot list 工作流
- 更好的 QA 检查项
- 更适合正文配图的例图
- README、安装说明、使用说明和翻译
- 针对常见失败结果的修正建议

## 风格要求

请保持默认主角为 Specimen 0：

- 米白身体
- 深色头顶、后背和尾巴斑块
- 明黄色大眼
- 黑色细竖瞳
- 三角耳
- 粗黑手绘轮廓
- 白底手账贴纸感
- 呆滞、松弛、像在认真发呆的幽默表情

请避免：

- 把猫画成圆滚滚 Q 版
- 换成完全不同的猫种或角色
- 加入复杂背景、商业海报构图或大段说明文字
- 直接复制参考图、原项目示例图或他人未授权作品
- 让猫猫只作为角落装饰，而不参与核心动作

## 示例图要求

提交示例图时，建议同时提供生成用的 prompt，并检查：

- 画面是否为 16:9 正文配图
- 猫猫是否承担核心动作
- 眼睛是否是黄眼细竖瞳
- 尾巴是否从身体后侧自然连出
- 中文标注是否简短、可读、无明显错字
- 画面是否不像 PPT 信息图或课程页

## 提交建议

小改动可以直接提交 PR。涉及默认风格、角色设定、授权说明的大改动，请在 PR 描述里说明原因、影响范围和示例效果。
