# 猫猫 IP

## 角色定义

猫猫是这套正文配图的固定视觉 IP。默认主角不是“任意猫”，而是 Specimen 0：一只白底深色斑块的双色短毛猫，拥有明黄色大眼和黑色细竖瞳，表情介于“极度清醒的呆滞”和“松弛发呆”之间。

## Specimen 0 标准设定

### 花色

- 身体主色是米白或暖白。
- 头顶到双耳附近有不规则深色斑块，颜色可为深褐、黑灰、橄榄棕黑。
- 后背有一块不规则深色斑。
- 尾巴大部分是深色，尾巴可以有轻微深浅色段。
- 脸部保留大面积白色，让黄色眼睛和深色头顶形成强识别。

### 眼睛

- 眼睛是角色灵魂，必须优先画准。
- 虹膜是明黄色、暖黄色或淡橄榄黄。
- 瞳孔是非常细的黑色垂直竖线，不是圆点，不是大黑眼珠。
- 眼睛可以圆、圆角椭圆、斜长椭圆，但必须有“清空大脑、看透一切”的呆滞幽默感。
- 不要星星眼、高光大眼、可爱少女眼。

### 脸部

- 鼻子是极小淡粉点或短小粉色形状。
- 嘴巴通常隐藏；只有舔爪、尴尬、清洁动作时露出柔和粉色小舌头。
- 胡须每侧 2-3 根，黑色短直线，简单有力。
- 耳朵是三角耳，内部浅粉，可有少量粉色短线。

### 身体

- 不是圆球猫，也不是大头娃娃猫。
- 身体可以拉长、趴下、坐成软三角、侧躺、半蹲、低头舔爪。
- 四肢简化但要保留猫的方向感：前爪可挡嘴、扒拉纸片、按住观点；后腿可以短而笨拙。
- 尾巴粗而流畅，可拖地、弯曲、绕物、指向、连接。

## 性格与动作

Specimen 0 的气质是：呆、醒、松弛、认真、略微荒唐。

常见动作：

- 伸爪把“想法纸片”摊开。
- 用尾巴把散乱观点圈起来。
- 舔爪，但眼睛仍然盯着读者。
- 趴在判断路径上，让用户看清断点。
- 用爪子按住一个黑箱，再把里面的证据扒出来。
- 把 AI 输出叼到一边，把人的思考摊在中间。
- 坐在一堆便签旁边，竖瞳发呆。
- 用尾巴当尺子，量一段逻辑有没有断。

## 可选变体

只有在用户明确要求多只猫或群像时，才允许出现其他猫：橘猫、灰白猫、长毛猫、黑猫、三花猫。即便出现，也要保持同一 journal sticker style，并让 Specimen 0 作为主角或视觉中心。

## 禁止

- 不要随机生成泛猫。
- 不要把默认主角换成橘猫、纯白猫、黑猫、长毛猫，除非用户明确要求。
- 不要把眼睛画成黑豆眼、圆点眼、水汪汪大眼或高光萌眼。
- 不要把瞳孔画成圆形；必须是细竖线。
- 不要把身体画成圆滚滚 Q 版吉祥物。
- 不要让猫猫只是坐在角落里看。
- 不要复刻参考图片里的具体姿势；可以复用花色、眼神、线条、质感和角色气质。

## 判断标准

一眼看过去，应该能认出这是同一只白底深色斑块、黄眼竖瞳、呆滞清醒的猫。如果只是“某只可爱猫”，就不合格。
