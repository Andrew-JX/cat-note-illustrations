# 风格 DNA

## 一句话

治愈系手账便签风 + 现代简约漫画感：纯白或近白背景，中等偏粗的不均匀黑色手绘线，贴纸剪纸感轮廓，低饱和大地色平涂，色块里有极淡纸感/色铅笔颗粒，默认主角是 Specimen 0 奶牛双色猫，不是泛猫、不是圆润 Q 版吉祥物。

## 画风定位

这套风格是 Journal Sticker Style / Cute Cartoon Comic Style。它像白底手账贴纸或便签上的猫咪小漫画：线条有阻尼感，颜色温暖克制，角色有轻微的荒唐和呆滞幽默。

它不属于产品草图、工业线稿、水彩、像素、3D、商业矢量海报、儿童绘本或 PPT 信息图。正文配图时，猫咪负责把文章观点演出来，而不是变成普通装饰贴纸。

## 默认主角：Specimen 0

默认每张图优先使用同一只猫：Specimen 0。

- 品种/花色：奶牛双色短毛猫，也可轻微偏三花；身体主色是米白/暖白。
- 头部斑块：头顶、额头和双耳附近有不规则深色斑块，颜色是深褐、黑灰或橄榄棕黑。
- 身体斑块：后背有一块不规则深色斑，尾巴大部分为深色，尾尖或尾根可以有深浅变化。
- 眼睛：超大黄色虹膜，形状是圆形、圆角椭圆或斜长椭圆；瞳孔必须是细长黑色竖线。
- 神态：极度清醒的呆滞感，像“清空大脑但看透一切”。不要甜美卖萌。
- 嘴鼻：嘴巴通常隐藏；舔毛或表达尴尬时可以露出粉嫩小舌头；鼻子是极小淡粉点。
- 胡须：每侧 2-3 根黑色短直线，干脆、对称或轻微不对称。

其他猫型只有在用户明确要求“多只猫”“猫咪群像”“橘猫/黑猫/长毛猫”等场景时才启用。普通正文配图默认不要随机换猫。

## 线条

- 使用中等偏粗的黑色手绘墨线，带轻微不均匀和阻尼感，像中性笔/钢笔手绘。
- 线条闭合完整，色块被黑线包住，有贴纸剪纸感。
- 线条末端圆润，但整体轮廓不要变成圆球。
- 保留猫的明确结构：尖三角耳、拉长身体、短脚、弯尾巴、折角腿。
- 不要机械矢量线、产品草图线、细碎抖线、书法笔触。

## 形体

- 不使用大头娃娃比例，不要默认头巨大、身体短圆。
- Specimen 0 的身体可以趴长、侧躺、坐直、舔爪、低头检查、用爪子扒拉物件。
- 耳朵是明显三角形，内部浅粉，耳尖可以稍微圆但不能变圆耳。
- 眼睛要大、黄、呆，瞳孔是细竖线，这是最关键的识别点。
- 爪子极简，只用短线或小粉点暗示脚趾/肉垫。
- 尾巴粗而流畅，通常深色，可以作为连接线、指针、拖拽物、边界或承接路径。
- 姿态要有一点“猫正在认真但笨拙地做事”的松弛感。

## 上色与质感

- 低饱和度平涂为主，颜色柔和、温暖、不刺眼。
- 主色：米白、暖白、暖灰、浅咖、淡橘、橄榄棕、黑灰、深褐。
- 色块内部允许极淡颗粒感，像再生纸张或色铅笔轻轻平涂，不要死板数字纯色。
- 阴影极少，仅在耳朵内部、舌头、斑块边缘或身体局部用非常克制的同色系浅色区分。
- 毛发纹理用同色系短线/短竖线/短虚线，稀疏分布，不刻画真实毛流。
- 眼睛使用明黄色/暖黄色虹膜 + 黑色细竖瞳；不要星星高光和闪亮大眼。

## 背景与文字

- 背景保持纯白或近白，干净无复杂场景。
- 可以有极轻微纸感，但不要明显纸纹、复古纸、噪点背景或阴影。
- 正文配图可以有少量中文短标注，最多 3-5 处，每处 2-6 个字。
- 不要做 catalog/specification sheet、解剖图、邮票格子、英文说明板、角色设定页，除非用户明确要求。

## 必须

- 16:9 横版正文配图。
- 默认出现 Specimen 0 双色猫，且承担核心动作。
- 一张图只表达一个核心动作、结构、状态或隐喻。
- 主体占画面约 35%-60%，保留白底留白。
- 黄色大眼 + 黑色细竖瞳 + 深色头顶/后背/尾巴斑块必须明显。

## 绝对不要

- 不要泛猫随机生成，不要每次换成不同品种。
- 不要圆滚滚 Q 版、kawaii 吉祥物、大头娃娃猫、玩具猫。
- 不要写实猫、摄影猫、3D 猫、毛绒玩具猫。
- 不要精致商业矢量、儿童绘本、水彩、厚涂、像素、产品草图。
- 不要 PPT 信息图、正式流程图、课程页、密集英文标注、角色说明书。
- 不要复刻参考图的具体姿势或多猫排版；要复用的是角色 DNA，不是构图。

## 审美方向

要像一只固定角色猫在白底手账贴纸里认真处理一个观点：粗线、暖色、呆眼、松弛、轻微好笑、清爽可读。
