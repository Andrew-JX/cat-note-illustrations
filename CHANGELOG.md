# Changelog

- Added GitHub-facing usage examples for context engineering, harness engineering, and loop engineering.

- Clarified that broken, floating, two-part, or arrow-like tails are generation failures.
- Added open-source project docs: `CONTRIBUTING.md`, this changelog, and clearer README guidance.
